Developer Template Tab

This is a starter page to build a new tab. Follow these guidelines:

  1. Put shared styles in static/styles/theme.css and tab-specific tweaks in a small CSS file if needed.
  2. Use static/js/ui.js helpers for CSRF-aware fetch and a loading spinner.
  3. Back-end routes should live in a clear blueprint (e.g., app/query/blueprints/query_api.py).
  4. Keep prompts in .md files in your feature folder.

Example API call:

POST /api/query/ask

  "prompt": "What are the last 3 A1c values?",
  "model_id": "default"
