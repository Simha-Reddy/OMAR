Static JavaScript libraries
===========================

Last updated: 2025-11-07

Migrated from `static/lib/README.md`.

Available libraries
-------------------
Tabulator (table/grid)
- `tabulator.min.css` / `tabulator.min.js` — v6.3.0
- `tabulator-5.5.0.min.css` / `tabulator-5.5.0.min.js` — v5.5.0 for compatibility

Marked (Markdown parser)
- `marked.min.js`

Usage snippet
-------------
```html
<link href="/static/lib/tabulator.min.css" rel="stylesheet">
<script src="/static/lib/marked.min.js"></script>
<script src="/static/lib/tabulator.min.js"></script>
```

Original sources
----------------
- Tabulator: https://unpkg.com/tabulator-tables/
- Marked: https://cdn.jsdelivr.net/npm/marked/

Updating
--------
1. Download latest versions
2. Replace files under `static/lib/`
3. Adjust version references if needed
